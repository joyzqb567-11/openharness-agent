5. 新增 token 自动刷新

在 learning_agent.py (line 425)：

def _get_valid_tokens(self) -> CodexOAuthTokens:
    tokens = self._token_store.load()
    if tokens is None or not tokens.refresh_token:
        tokens = self._login_callback()
        self._token_store.save(tokens)
        return tokens
    if tokens.expires_at <= self._now_ms() + 60000:
        tokens = self._refresh_tokens(tokens)
        self._token_store.save(tokens)
    return tokens
意思是：没 token 就登录，快过期就自动刷新。
