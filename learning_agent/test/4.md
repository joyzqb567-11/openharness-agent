# Computer Use 场景路径迁移清单

本文件是给用户学习用的副本记录，不参与项目运行。

本次修改把 `learning_agent/acceptance_controller/scenarios/*.json` 中已经删除的旧包前缀：

```text
learning_agent.computer_use.
```

统一迁移为当前源码实际存在的 v2 包前缀：

```text
learning_agent.computer_use_mcp_v2.windows_runtime.
```

这样做的原因是 `learning_agent/computer_use` 目录当前已经不存在；如果场景继续引用旧包，真实可见终端验收或生产矩阵抽到这些场景时会直接出现 `ModuleNotFoundError`，压力测试结果就不可信。
