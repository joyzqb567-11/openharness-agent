# agent.py 教学视频视觉设计

## Style Prompt

技术教学风格，深色代码画布，重点用青绿色和金色高亮流程节点。画面要像一张清晰的工程白板，而不是炫技动画。每一幕只讲一个概念，文字少、层级清楚，让代码新手能跟得上。

## Colors

- `#0B1020`：主背景，像夜间代码编辑器，降低视觉噪音。
- `#F8FAFC`：主文字，保证中文说明清楚可读。
- `#38D9A9`：关键流程高亮，用来标出“请求正在走到哪里”。
- `#FBBF24`：风险和权限提醒，用来标出“这里需要确认”。
- `#94A3B8`：辅助说明文字，避免所有文字都抢注意力。

## Typography

- 中文优先使用 `Microsoft YaHei UI`，英文代码使用 `Consolas`。
- 标题粗一些，正文保持中等字重，避免密密麻麻。

## What NOT to Do

- 不做花哨渐变背景，避免抢走代码结构本身。
- 不把每一行代码都塞进画面，视频讲结构，讲义补细节。
- 不用过小字号，保证 1080p 播放也能读。
- 不制造大量闪动动画，教学视频以稳定理解为主。
