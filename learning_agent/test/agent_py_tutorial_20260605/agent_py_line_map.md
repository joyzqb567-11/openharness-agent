## 最建议先读的 6 个入口

1. `__init__()`：第 242-352 行。看 agent 保存了哪些状态。
2. `run_events()`：第 454-636 行。看用户请求如何变成一次完整运行。
3. `_build_initial_messages()`：第 1197-1220 行。看模型第一眼看到什么。
4. `_available_tool_schemas()` 相关：第 879-1009 行。看工具如何出现在模型面前。
5. `_execute_tool()` / `_execute_mcp_tool()`：第 1383-1542 行。看工具如何真正执行。
6. `_resolve_workspace_path()`：第 4147-4160 行。看文件读写为什么不会随便越界。

## 用一句话记住

`agent.py` 不是一个普通业务文件，它是 agent 的运行时总线。第 242-352 行保存状态，第 454-636 行组织主循环，第 879-1542 行管理和执行工具，第 1769-2202 行处理长任务，第 3399-3767 行接入桌面控制，第 4163-4229 行处理终端权限和启动入口。
