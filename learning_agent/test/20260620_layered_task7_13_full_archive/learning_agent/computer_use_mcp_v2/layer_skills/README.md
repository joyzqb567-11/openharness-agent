# Computer Use Internal Layer Skills

These files are internal Computer Use layer skills only.

They are not global Codex skills, not OpenHarness global agent prompts, and not generic project prompts.

The Computer Use runtime may load these small prompts to improve a specific layer:

- intent understanding
- stage planning
- structured observation
- verification
- reflection and learning
- deterministic batch execution contract

The executor must stay deterministic and target-bound. Prompt files guide understanding, planning, observation interpretation, verification, and reflection. They must not authorize free-form low-level desktop action dispatch.
